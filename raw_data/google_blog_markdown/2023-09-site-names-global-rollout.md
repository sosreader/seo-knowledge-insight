# Site names on Google Search now available for all languages
- **發佈日期**: 2023-09-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2023/09/site-names-global-rollout
- **來源類型**: article
- **來源集合**: google-search-central
---
Thursday, September 7, 2023

Last October, [site names](/search/docs/appearance/site-names) were
[introduced on mobile results](/search/blog/2022/10/introducing-site-names-on-search)
for select languages, which [expanded to desktop in March](https://twitter.com/googlesearchc/status/1633535806915891200).
Today, we're happy to announce that site names are now available in all languages where
Google Search is available, on both mobile and desktop. Site names are supported for domains
(for example, `example.com`) and subdomains (for example, `subdomain.example.com`)
but not for subdirectories (for example, `example.com/subdirectory`).

This global support began rolling out slowly over the past few weeks and now is complete.
Should you have any questions, please see our [documentation about site names](/search/docs/appearance/site-names)
and feel free to post in the [Search Central help community](https://support.google.com/webmasters/thread/227739087).
