# Sunsetting the generic rich results search appearance on Search Console
- **發佈日期**: 2021-05-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2021/05/rich-results-search-appearance-sunsetting
- **來源類型**: article
- **來源集合**: google-search-central
---
Tuesday, May 25, 2021

Over the last few years, we have been hard at work adding more and more specific rich results search appearances to the Search Console Performance
reports. Today we have 19 [supported search
appearance types](https://support.google.com/webmasters/answer/7576553#by_search_appearance) to help website owners understand how their websites perform in Google Search.

A few years ago we created a generic "Rich results" search appearance group that included multiple rich result types, including those that didn't
have their own breakout row on Search Console. This is not required anymore, since we now have breakouts for most rich result types, for example:
[Event](/search/docs/appearance/structured-data/event), [FAQ](/search/docs/appearance/structured-data/faqpage),
[HowTo](/search/docs/appearance/structured-data/how-to), and more.

In addition, we realize that grouping data for rich results is not ideal, as each type may have a significantly different behavior. Therefore we decided to
sunset the Rich result search appearance.

![Rich results search appearance to be deprecated](/static/search/blog/images/rich-results-search-appearance.png "Rich results search appearance to be deprecated")

**Notes:**

* The removal of the search appearance does not affect your traffic in any way; it is only a reporting change.
* We will keep supporting the search appearance until August 1, 2021. Following that date, it will no longer be available in Search Console or the API.

We’ll keep investing in adding more rich result types to Search Console, and hope the reports will continue being a source of insights for you.

If you have any questions or concerns, please reach out on the [Google
Search Central Community](https://support.google.com/webmasters/threads?thread_filter=(category:search_console)) or on [Twitter](https://twitter.com/googlesearchc).
