# Search Console 新增關聯功能透過集合功能整理內容你可以依據偏好儲存及分類內容。
- **發佈日期**: 2021-02-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2021/02/search-console-associations?hl=zh-tw
- **來源類型**: article
- **來源集合**: google-search-central-zh
---
2021 年 2 月 22 日 (星期一)

我們很高興在此宣布，Search Console 已在設定選單中新增更強大的[關聯頁面](https://search.google.com/search-console/settings/associations?hl=zh-tw)。「關聯」是指 Search Console 資源和其他 Google 服務中實體或資源之間的連結，例如 Search Console 資源與 Google Analytics (分析) 帳戶、YouTube 頻道或 Chrome 線上應用程式商店帳戶之間的連結。

現有的關聯會全數自動轉移至全新的「關聯」頁面。
進一步瞭解[如何使用關聯功能](https://support.google.com/webmasters/answer/9419894?hl=zh-tw)。

![Google Search Console 關聯頁面](https://developers.google.com/static/search/blog/images/search-console-associations.png?hl=zh-tw "Google Search Console 關聯頁面")

只要將網站與下列 Google 服務建立關聯，即可讓網站的功能更加多元。Search Console 支援與下列項目建立關聯：

* [Google Analytics (分析)](https://support.google.com/analytics/answer/1308621?hl=zh-tw) 資源：可在 Google Analytics (分析) 資源中查看 Search Console 的資料，例如自然搜尋查詢。即日起，您還可以將 Search Console [網域資源](https://support.google.com/webmasters/answer/34592?hl=zh-tw)連結至 Google Analytics (分析)，藉此獲得涵蓋範圍更廣的自然搜尋結果。建議您採取這個做法，以便在 Google Analytics (分析) 的 [Search Console 報表](https://support.google.com/analytics/answer/1308626?hl=zh-tw)中獲得更完整的結果。
* [Google Ads](https://support.google.com/google-ads/answer/7529561?hl=zh-tw) 帳戶：可在 Google Ads 中查看付費和自然搜尋報表。
* [YouTube](https://support.google.com/youtube/answer/2887282?hl=zh-tw) 頻道：可將影片連結至關聯網站。
* [Play 管理中心](https://support.google.com/googleplay/android-developer/answer/6041489?hl=zh-tw)應用程式：可使用應用程式索引功能。
* [Actions 主控台](https://developers.google.com/assistant/console/brand-verification?hl=zh-tw)專案：可驗證您的品牌並啟用跨平台功能。
* [Chrome 線上應用程式商店](https://support.google.com/chrome/a/answer/2714278?hl=zh-tw)帳戶：能以您網站的名義在 Chrome 線上應用程式商店中發布應用程式和擴充功能。

如對全新的「關聯」頁面有任何問題或意見，歡迎前往[搜尋中心說明論壇](https://support.google.com/webmasters/go/community?hl=zh-tw)提出，或是透過[其他支援管道](https://developers.google.com/search/help?hl=zh-tw)與我們聯絡。此外，也歡迎參閱[關聯功能說明頁面](https://support.google.com/webmasters/answer/9419894?hl=zh-tw)，進一步瞭解如何將 Search Console 資源與其他 Google 服務建立關聯。
