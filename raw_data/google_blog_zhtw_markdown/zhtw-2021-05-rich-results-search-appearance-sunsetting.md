# Search Console 即將停止支援一般複合式搜尋結果的搜尋外觀
- **發佈日期**: 2021-05-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2021/05/rich-results-search-appearance-sunsetting?hl=zh-tw
- **來源類型**: article
- **來源集合**: google-search-central-zh
---
2021 年 5 月 25 日，星期二

過去幾年來，我們一直努力在為 Search Console 的成效報表加入更加具體的複合式搜尋結果搜尋外觀，目前已經有 19 種[支援的搜尋外觀](https://support.google.com/webmasters/answer/7576553?hl=zh-tw#by_search_appearance)能夠協助網站擁有者，讓他們瞭解自家網站在 Google 搜尋中的成效。

我們在幾年前建立了通用的「複合式搜尋結果」搜尋外觀群組，其中包含多種複合式搜尋結果，包括許多在 Search Console 中沒有明細資料列的複合式搜尋結果類型。不過，由於現在大多數類型的複合式搜尋結果 (例如[活動](https://developers.google.com/search/docs/appearance/structured-data/event?hl=zh-tw)、[常見問題](https://developers.google.com/search/docs/appearance/structured-data/faqpage?hl=zh-tw)、[操作說明](https://developers.google.com/search/docs/appearance/structured-data/how-to?hl=zh-tw)等) 都已經有單獨的明細資訊，所以這個群組已經沒有存在的必要了。

此外，我們也瞭解將複合式搜尋結果的資料分成一組不太合適，因為每種類型結果的機制可能大不相同。因此，我們決定淘汰複合式搜尋結果的搜尋外觀。

![即將淘汰的複合式搜尋結果搜尋外觀](https://developers.google.com/static/search/blog/images/rich-results-search-appearance.png?hl=zh-tw "即將淘汰的複合式搜尋結果搜尋外觀")

**注意：**

* 移除搜尋外觀不會影響您的流量；這只是報表上的調整。
* 我們將支援此搜尋外觀到 2021 年 8 月 1 日，之後 Search Console 和 API 就不會再提供相關資訊。

我們會繼續努力在 Search Console 中加入更多種複合式搜尋結果，希望相關報表日後也能持續做為您的深入分析來源。

如果您有任何問題或疑慮，歡迎前往 [Google 搜尋中心社群](https://support.google.com/webmasters/threads?thread_filter=%28category%3Asearch_console%29&hl=zh-tw)或 [Twitter](https://twitter.com/googlesearchc) 尋求協助。
