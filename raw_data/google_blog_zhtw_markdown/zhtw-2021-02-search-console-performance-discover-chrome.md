# Search Console 的探索報表已提供 Chrome 資料
- **發佈日期**: 2021-02-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2021/02/search-console-performance-discover-chrome?hl=zh-tw
- **來源類型**: article
- **來源集合**: google-search-central-zh
---
2021 年 2 月 2 日星期二

很多使用者都會利用探索功能及時獲得自身喜愛主題的最新資訊，甚至就連沒有特別搜尋目標時，也會瀏覽探索專區顯示的內容。現在發布者和網站擁有者都可以查看[探索成效報表](https://support.google.com/webmasters/answer/9216516?hl=zh-tw)並檢閱其中的重要指標，瞭解自家網站在 Android 版或 iOS 版 Google 應用程式中「探索」專區的成效。

除此之外，當使用者在 Android 版或 iOS 版的 Chrome 中開啟新分頁時，也同樣會接觸到探索內容。只不過，探索成效報表原先並未採計來自 Chrome 的探索流量。

如今，我們已經在 Search Console 開闢探索統計資料專區，您可以在這裡一覽網站的所有探索曝光和點擊資料，其中也包括來自 Chrome 的流量資料。

![Google Search Console 探索成效報表](https://developers.google.com/static/search/blog/images/search-console-discover-performance-report.png?hl=zh-tw "Google Search Console 探索成效報表")

在接下來幾個月，您就可以在 Search Console 的探索成效報表中逐漸看到網站的相關資料，而這也意味著回報的網站流量可能會較先前增加。

我們會從少部分開始逐步為來自 Chrome 的探索流量使用新的[來源](https://web.dev/articles/same-site-same-origin?hl=zh-tw#origin)參照網址 `https://www.google.com/`，以便與來自 Google 應用程式的探索流量保持一致。原先的參照網址則是 `www.googleapis.com/auth/chrome-content-suggestions`。

如果您對這份報表有任何疑問或意見，歡迎前往搜尋中心[說明論壇](https://support.google.com/webmasters/go/community?hl=zh-tw)提出，或是透過[其他管道](https://developers.google.com/search/help?hl=zh-tw)與我們聯絡。此外，也歡迎瀏覽 [Google 探索和您的網站](https://developers.google.com/search/docs/advanced/mobile/google-discover?hl=zh-tw)頁面，進一步瞭解探索內容可能出現的方式和其他參考提示。
