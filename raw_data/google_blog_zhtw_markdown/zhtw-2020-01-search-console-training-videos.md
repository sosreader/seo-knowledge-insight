# Search Console 訓練課程系列影片正式推出
- **發佈日期**: 2020-01-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2020/01/search-console-training-videos?hl=zh-tw

2020 年 1 月 15 日，星期三

我們在 YouTube 上推出了有關 [Google Search Console](https://goo.gle/searchconsole) 的全新系列影片！影片重點在於介紹如何運用 Search Console 改進網站，進而在 Google 搜尋服務中獲得更佳成效。我們將一一介紹您可以使用的報告，並舉例說明在哪裡可以找到資料、如何分析資料，以及如何解決可能會影響搜尋結果呈現的問題。我們將發布十多集影片，主要介紹 Search Console 提供的大部分功能。

早在 2019 年將新版 Search Console [遷移完成](https://developers.google.com/search/blog/2019/09/goodbye-old-search-console?hl=zh-tw)時，我們就知道，詳盡的訓練課程影片，可協助使用者瞭解這項產品與許多相關用途。下方是已經發布的影片，日後還會陸續推出更多影片！歡迎查看 [Search Console 訓練課程播放清單](https://goo.gle/sct)，我們每兩週會推出一部新影片。您也可以訂閱 [Webmasters YouTube 頻道](https://www.youtube.com/channel/UCWf2ZlNsCGDS89VBF_awNvA?sub_confirmation=1&hl=zh-tw)，新影片上傳時就能收到通知。

希望在本系列影片結束時，您會覺得 Search Console 的資料值得探究，令人期待。如果您有任何想法，歡迎留言給我們，或在 [Twitter](https://twitter.com/googlesearchc) 標記我們。
