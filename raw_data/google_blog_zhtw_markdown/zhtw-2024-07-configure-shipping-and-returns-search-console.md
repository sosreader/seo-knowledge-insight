# 直接在 Search Console 中設定運送和退貨資訊
- **發佈日期**: 2024-07-01
- **作者**: Google Search Central
- **來源 URL**: https://developers.google.com/search/blog/2024/07/configure-shipping-and-returns-search-console?hl=zh-tw
- **來源類型**: article
- **來源集合**: google-search-central-zh
---
2024 年 7 月 11 日，星期四

消費者購物時，會希望瞭解購買產品的總價。消費者在線上購買產品時，運送速度、費用和退貨政策是主要考量因素。不過對某些商家來說，設定運送與退貨服務可能會相當困難，而且可能錯失為目標客群提供所需資訊的良機。

我們去年開始支援[產品層級的運送和退貨標記](https://developers.google.com/search/blog/2023/04/shipping-and-returns-information?hl=zh-tw)。我們最近[新增了標記支援，方便您的商家指定一般退貨政策](https://developers.google.com/search/blog/2024/06/structured-data-return-policies?hl=zh-tw)，不必為您銷售的每項產品分別定義政策。我們很高興在此宣布，您可以更輕鬆地在 Search Console 中直接新增運送或退貨資訊。如需操作步驟，請參閱[說明中心文章](https://support.google.com/webmasters/answer/14907594?hl=zh-tw)。

**注意**：Search Console 的運送和退貨設定，優先順序高於網站上的設定，包括產品層級的商家資訊標記。進一步瞭解[優先順序原則](https://developers.google.com/search/docs/appearance/structured-data/merchant-listing?hl=zh-tw#precedence)。

![Search Console 中顯示運送和退貨設定](https://developers.google.com/static/search/blog/images/shipping-and-returns-search-console.png?hl=zh-tw)

完成[說明中心文章](https://support.google.com/webmasters/answer/14907594?hl=zh-tw)所述的步驟後，您可以直接在 Search Console 中管理 Merchant Center 帳戶層級的運送和退貨政策。這項設定會自動出現在 [Merchant Center](https://www.google.com/retail/solutions/merchant-center/?hl=zh-tw)。

只要提供正確且最新的運送和退貨資訊，就能使您在 Google 搜尋結果中的產品資訊更加豐富。您可以藉由在產品資訊旁展示出貨服務(例如[免運費及快速出貨](https://support.google.com/merchants/answer/14264257?hl=zh-tw))，提高曝光度和潛在流量。

希望這些新增項目可進一步協助您改善在搜尋結果中的呈現內容。如有任何問題或疑慮，請透過 [Google 搜尋中心社群](https://support.google.com/webmasters/threads?thread_filter=%28category%3Asearch_console%29&hl=zh-tw)、[Twitter](https://twitter.com/googlesearchc) 或 [LinkedIn](https://www.linkedin.com/showcase/googlesearchcentral/) 與我們聯絡。
