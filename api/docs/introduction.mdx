# SEO Knowledge Insight API

SEO knowledge base API with RAG chat, semantic search, weekly reports, and ETL pipeline management.

## Features

- **QA Knowledge Base** -- 1,300+ SEO Q&A pairs from meetings and articles
- **Semantic Search** -- Hybrid search (embedding + keyword) with synonym expansion
- **RAG Chat** -- Single-turn RAG and multi-turn agent mode
- **Weekly Reports** -- Automated SEO report generation from metrics snapshots
- **Pipeline Management** -- ETL pipeline for Notion, Medium, iThome, and Google Case Studies
- **Session Management** -- Persistent multi-turn chat sessions

## Quick Start

```bash
# Health check (no auth required)
curl http://localhost:8002/health

# Search the knowledge base
curl -X POST http://localhost:8002/api/v1/search \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-key" \
  -d '{"query": "Core Web Vitals", "top_k": 5}'
```
